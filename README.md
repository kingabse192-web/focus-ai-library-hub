# Focus & AI Library Hub 🎵📚

A personalized Focus & AI Library Hub — a digital study companion with a retro mixtape deck, Pomodoro timer, AI study assistant, and cozy candlelight mode.

## Features

- **9 Jazz Tracks + Custom Playlist** — Miles Davis, Chet Baker, Bill Evans, and more
- **Retro Cassette Tape Deck** — Analog crackle, spinning reels, warm vibes
- **Pomodoro Timer** — 25m / 5m / 50m with focus countdown
- **Gemini AI Assistant** — Live real-time study Q&A
- **Meberat Tefta (Candlelight Mode)** — Cozy theme for power-cut nights
- **Deep Focus Mode** — Fullscreen, Wake Lock, DND, distraction-free

## How to Run Locally

```bash
git clone https://github.com/kingabse192-web/focus-ai-library-hub.git
cd focus-ai-library-hub
```

Then open `index.html` in any modern web browser.

**Note:** For the Gemini AI assistant to work locally, click the settings gear ⚙️ and paste your Gemini API key.
